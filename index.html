About me

I am a Computer Engineering professional with a Labor Technician degree in Computer Systems and a strong passion for building reliable software systems and modern web applications. My work focuses on Full-Stack development combined with practical DevOps practices, allowing me to design, build, deploy, and maintain applications across the entire development lifecycle. On the backend, I work primarily with Java (Spring Framework and Spring Boot) and Python, building APIs, implementing business logic, integrating services, and managing application architecture. I have experience working with relational databases such as MySQL, designing efficient data models and ensuring reliable data management within production environments. On the frontend, I build responsive and user-focused interfaces using JavaScript, HTML5, and CSS, focusing on clean structure, usability, and cross-device compatibility. In addition to application development, I have a strong interest in infrastructure and DevOps practices. I work with Linux server environments, configure Nginx web servers and reverse proxies, and implement CI/CD pipelines using Git and GitHub Actions to automate deployment workflows. This experience allows me to understand not only how to build applications, but also how to run them reliably in production. I believe that modern developers should understand both software development and the infrastructure that supports it. For this reason, I continuously work on improving my knowledge in system architecture, deployment automation, monitoring, and scalable web infrastructure. I am motivated by solving real problems through technology and by building systems that are practical, maintainable, and scalable. My goal is to contribute to impactful projects, collaborate with strong technical teams, and continue growing as a Full-Stack developer with DevOps and infrastructure expertise.
